>##暂时是一个提供给广东工业大学(GDUT)学子查询教务系统成绩的App
>
>![Logo](https://raw.githubusercontent.com/gdutrdc/GDUTHelper/dev/app/src/main/ic_launcher-web.png) 
 
###已完成的功能有

* 个人课程表
     * 查看
     * 自定义背景
* 查询成绩
* 查询考试时间（以及桌面控件）
* 查询个人信息
* 教学质量评价（试验性）
* 查询教务系统上的个人信息
* 等级考试查询


###*待完成的功能*

* 课程表的
     * 桌面控件
     * 额外课程添加
